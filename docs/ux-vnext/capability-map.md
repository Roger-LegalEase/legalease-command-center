# CCX-001 capability map

Source SHA: `8812c2f31328cf1e8e36d36efc22bac55e1f0498`

This is an inventory of meaningful user capabilities already present in the legacy
Command Center. It is not a promise that every capability is complete or production
connected. “Unavailable,” “read-only,” “manual,” and “approval only” states remain
part of the truth contract.

Unless a row says otherwise, the current hash shell is readable by authenticated
owner/admin/operator roles, full-state data is owner/admin-only, viewer access is
aggregate-report-only, and every write is rechecked by the existing endpoint role and
capability layer. CCX-001 adds no endpoint and calls no provider.
