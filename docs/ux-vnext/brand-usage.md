# vNext brand usage

The approved direction is `docs/ux-vnext/reference/command-center-vnext-approved-direction.png`.

- Use the navy shell and soft teal selection treatment defined by the vNext brand contract.
- Primary actions use exact orange `#F04800`.
- The shell uses the exact official white-logo asset: `assets/brand/logos/legalease-logo-white-2025.png`.
- Do not recolor, trace, redraw, crop, filter, add effects to, or substitute the official logo. Preserve its aspect ratio and clear space.
- Do not infer a new brand color from screenshots. Validate desktop through mobile against the coded tokens and automated contract.

The reference image guides composition. The checked-in official logo and coded tokens remain the implementation source of truth.
