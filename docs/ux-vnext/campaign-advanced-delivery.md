# CCX-410 Advanced Campaign delivery detail

Advanced delivery is closed by default and starts with a plain-language connection summary. Authorized detail includes delivery tracking status, domain verification, bounce and complaint limits, aggregate suppression detail, SendGrid event health, batch detail, and bounded opaque event references for diagnostic roles.

Unavailable telemetry remains unavailable. The projection omits recipient content, provider payloads, credentials, tokens, and secrets. It exposes no edit or override capability. A provider connection is reported separately from sending enablement and never implies authority to send.
