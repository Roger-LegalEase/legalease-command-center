# Founder Language Guide

Use founder language in normal UI. The main nav must be:

- Today
- Work
- Social
- Proof
- Search

Settings stays secondary.

Avoid these terms in normal founder UI:

- RCAP
- Production Activation
- Operating Memory
- Operator
- Live Gates
- API status
- OAuth
- token
- webhook
- schema
- generated client
- diagnostics
- route map

Preferred replacements:

- Operating Memory -> Notes & Decisions
- OS Health -> App Status
- Data Integrity -> Data Check
- Smoke Test -> Self-Check
- Safe Mode -> Recovery Mode
- Handoff Contract -> Handoff Notes
- Live Gates -> Publishing is off

Keep normal pages short, practical, and action-oriented.
