# Product Contract

LegalEase Command Center is Roger's internal solo-founder operating system. It is not the Partner Journey app.

The accepted main workflow is:

Today / Work / Social / Proof / Search

The daily founder workflow is:

Today -> Capture -> Tasks -> Decisions & Blockers -> Daily Closeout -> Tomorrow Plan

The Social workflow is:

Post Idea -> Draft -> Preview -> Ready to Publish -> Publish manually -> Mark published manually

The app must never do these without explicit future approval and separate live gates:

- send emails
- create live social posts
- write calendars
- publish pages
- activate dashboards
- enable live gates
- run external actions
- change the Partner Journey repo or call Partner Journey APIs

Normal UI uses founder language. Do not show code-speak, route names, schemas, diagnostics, provider tokens, or internal architecture on Today, Work, Social, Proof, or Search.

Social is first-class. Do not remove, hide, bury, or de-prioritize Social.
